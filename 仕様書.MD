# ThinkFlow Pro - 仕様書 (Infinite Canvas Edition)

## 1. プロジェクト概要
ThinkFlow Proは、ユーザーが思考を構造化し、視覚的に整理するためのWebベースのダイアグラム作成ツールです。カテゴリー分けされた要素（ノード）を自由に配置し、それらを論理的なラインで接続することで、複雑なアイディアを可視化します。

## 2. 技術スタック
- **Frontend**: HTML5, CSS3 (Tailwind CSS), JavaScript (Vanilla JS)
- **Libraries**:
  - [LeaderLine (New)](https://github.com/anseki/leader-line): 要素間の動的な接続線の描画
  - [html2canvas](https://html2canvas.hertzen.com/): キャンバスの画像化
  - [jsPDF](https://rawgit.com/MrRio/jsPDF/master/docs/index.html): 画像化したキャンバスのPDF化
- **Storage**: Browser LocalStorage (自動保存および復元)

## 3. 主要機能

### 3.1 無限キャンバス (Infinite Canvas)
- **ズーム機能**: マウスホイールによってキャンバスを 20% 〜 300% の範囲で拡大・縮小可能。
- **パン機能**: 右クリック（または Ctrl + 左クリック）ドラッグにより、キャンバス全体を自由に移動可能。
- **視点リセット**: ボタン一つで等倍（100%）および初期位置（0, 0）へ復帰。

### 3.2 ノード（要素）管理
- **作成/編集**: テキストエリアに内容を入力し、ボタンクリックでノードを生成。既存ノードをクリックして内容を即時変更可能。
- **カテゴリー分類**: 6種類の意味（資産・現実・感情・推移・仕組み・理想）に基づき色分けが可能。
- **表示モード切り替え**: 「枠あり（Box）」と「枠なし（Text Only）」を切り替え、強調度を調整。
- **リサイズ**: 右下のハンドルをドラッグすることで、要素の大きさを自由に変更可能。
- **削除**: ノードを右クリックすることで即時削除（接続されている線も自動削除）。

### 3.3 接続（コネクト）システム
- **動的ライン**: 2つの要素を順にクリックすることで、自動的に流体（Fluid）パスで接続。
- **矢印スタイル**: 「順方向（A ➔ B）」と「矢印なし（A ― B）」を選択可能。
- **削除機能**: 描画されたライン自体を直接クリックすることで削除。
- **自由描画**: キャンバス上の何もないところをドラッグすることで、直線的な補助線を描画可能。

### 3.4 ユーティリティ
- **Undo (元に戻す)**: 最大30ステップまでの操作履歴を保持し、Ctrl+Zまたはボタンで復元。
- **エクスポート**: PDF形式での出力、およびテキスト形式によるデータの保存/読込に対応。
- **自動保存**: 操作のたびにLocalStorageへ状態を保存し、リロード時に復元。

## 4. 操作マニュアル

| 操作項目 | 方法 |
| :--- | :--- |
| **画面の拡大・縮小** | マウスホイールを回す |
| **画面の移動（パン）** | 右クリックしながらマウスを動かす |
| **要素の追加** | テキスト入力後「要素を追加」ボタンを押す |
| **要素の移動** | 要素を左クリックで掴んでドラッグ |
| **要素間の接続** | 1つ目の要素をクリック ➔ 2つ目の要素をクリック |
| **要素の削除** | 要素の上で右クリック |
| **接続線の削除** | 黒い線（パス）の上を左クリック |
| **自由な線を引く** | キャンバスの背景部分を左ドラッグ |

## 5. データの構造
保存されるデータは以下のJSON形式で管理されます。
- `nodes`: ノードのID、座標、サイズ、テキスト、カテゴリー、表示タイプ
- `leaderLines`: 接続元のID、接続先のID、線のタイプ
- `freeLines`: 自由描画されたSVGパスの座標群
